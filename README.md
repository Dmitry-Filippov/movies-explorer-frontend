# movies-explorer-frontend
Фронтенд дипломного проекта web факультета Yandex.Practicum   
Проект позволяет создать профиль и осуществлять поиск по фильмам, лайкать фильмы, фильтровать по длительности

### quick start   
`npm install`  
`npm start`
